## Command Line

```text
start  Online
====

start [--store value] [--port value] [--host] [--pkey] [--root-email] [--clean]

    * --store / string / mongodb://localhost/trop_api, URL refers to
      MongoDB server
    * --port / integer / 80, port to listen
    * --host / string / 0.0.0.0, host to bind on
    * --pkey / string, path to private key file
    * --root-email / string, root username, it will be create if does not
      existed
    * --clean, clean all of database
```
